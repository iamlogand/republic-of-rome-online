# Republic of Rome Online

An online adaptation of the board game [The Republic of Rome](https://boardgamegeek.com/boardgame/1513/republic-rome). Players control rival factions of Roman senators, cooperating to prevent Rome's collapse while competing to become the most powerful faction — if Rome falls, everyone loses. See `README.md` at the repo root for the regularly updated feature checklist.

## Directories

- **backend** — Django REST API serving game logic and state
- **frontend** — Next.js app providing the player-facing UI
- **game-data** — structured documentation for the original board game rules and components
- **rorcli** — CLI tool for querying the game-data rules and components
