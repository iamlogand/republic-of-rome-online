# Terminology changes

This document explains terminology used in this implementation that differs from the original board game.

## Below minimum force → Risky command

The term "below minimum force" has been replaced with "risky command". Since players no longer need to do the combat calculations themselves, a more thematic term makes sense.

## Drought → Famine

The term "drought" has been replaced with "famine". Droughts are natural occurrences that aren't caused by humans. However, in this game, warfare can trigger a drought. Famines are more fitting since they were often caused by droughts or war.
