# Concession Cards {#concession}

[← Components Index](README.md)

## Armaments {#concession-armaments}

- Immediately collect 2T per new Legion raised.
- May be destroyed by a Natural Disaster Event.
- Return to Curia if Destroyed.
- Return to Forum if owning Senator is convicted or killed.

## Ship Building {#concession-ship-building}

- Immediately collect 3T per new Fleet raised.
- May be destroyed by a Natural Disaster Event.
- Return to Curia if Destroyed.
- Return to Forum if owning Senator is convicted or killed.

## Egyptian Grain {#concession-egyptian-grain}

- 5T / Turn.
- Voluntary x2 Grain Concession Income and −2 Popularity if Drought in effect. +1 Income multiplier and −1 Popularity per additional Drought effect.
- Destroyed by Alexandrine War.
- Return to Curia if Destroyed.
- Return to Forum if owning Senator is convicted or killed.

## Sicilian Grain {#concession-sicilian-grain}

- 4T / Turn.
- Voluntary x2 Grain Concession Income and −2 Popularity if Drought in effect. +1 Income multiplier and −1 Popularity per additional Drought effect.
- Destroyed by Sicilian Slave Revolts.
- Return to Curia if Destroyed.
- Return to Forum if owning Senator is convicted or killed.

## Mining {#concession-mining}

## Land Commissioner {#concession-land-commissioner}

- 3T / Turn.
- Cannot be Destroyed.
- Return to Forum if owning Senator is convicted, killed or no Land Bill is in effect at end of Forum Phase.

## Tax Farmer 1 {#concession-tax-farmer-1}

- Destroyed on a 1 on 1d6 as a result of Punic Wars, Hannibal, Gladiator Revolt and/or Spartacus.
